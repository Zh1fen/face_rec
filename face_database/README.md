# 人脸数据库目录

请在此目录中按以下结构组织人脸图片：

```
face_database/
├── 张三/
│   ├── photo1.jpg
│   ├── photo2.jpg
│   └── photo3.jpg
├── 李四/
│   ├── photo1.jpg
│   └── photo2.jpg
└── 王五/
    └── photo1.jpg
```

## 注意事项

1. **文件夹命名**: 每个子文件夹的名称就是该人员的姓名
2. **图片格式**: 支持 .jpg, .jpeg, .png, .bmp, .tiff 格式
3. **图片质量**: 请确保人脸清晰、正面、光线充足
4. **图片数量**: 建议每人至少 2-3 张不同角度的照片
5. **文件大小**: 建议每张图片不超过 5MB

## 图片要求

- ✅ 人脸清晰可见
- ✅ 正面或接近正面角度
- ✅ 光线充足
- ✅ 没有遮挡（墨镜、帽子等）
- ✅ 表情自然
- ❌ 避免模糊、过暗的照片
- ❌ 避免侧脸或仰头低头的照片

组织好图片后，运行以下命令构建特征数据库：

```bash
python build_database.py
```
